#  Personal Portfolio

A responsive and modern **personal portfolio website** showcasing my skills, projects, and professional journey as a developer.  
Built to highlight my work and technical expertise in a clean and user-friendly manner.

---

## About the Project

This portfolio website serves as a **digital resume** and professional profile.  
It provides an overview of who I am, the technologies I work with, the projects I have built, and how to contact me.

The goal of this project is to create a strong online presence and demonstrate frontend development skills, UI/UX design, and best coding practices.

---

##  Objectives

- Showcase personal projects and skills
- Create a professional online identity
- Build a responsive and visually appealing website
- Improve frontend development and design skills
- Make information easily accessible to recruiters

---

##  Features

-  Home section with introduction
-  About Me section
-  Skills & Technologies
- Projects showcase
-  Resume download
-  Contact form / contact details
-  Fully responsive design
-  Clean and modern UI
